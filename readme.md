## 프로젝트 개요

- 자신만의 외국어 단어장을 제작 및 학습할 수 있는 외국어 학습 웹 어플리케이션
- React.js 를 이용한 프론트엔드 개발
- Express.js 와 MongoDB를 이용한 백엔드 개발
